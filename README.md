# Call Log Data Analysis

**Project:** CalSAWS (California Statewide Automated Welfare System) Call Center of the county

**Consumer:** Contra Costa County EHSD

**Program languages:**  Python, Jupyter Notebook

**Description:**   Analysis of call center data using statistical methods to identify characteristic distributions and correlations. The repository contains some files with my code and project description:

 - call_log.ipynb - Python notebook with source code
 - Part_1_data_analysis.docx - project description, part 1
 - Part_2_correlation_study.docx - project description, part 2
 
**Results / Key Findings:** The analysis revealed that the source data follow a normal distribution, and a linear regression ML model can be used as a first approximation for predicting the number of calls, although it is desirable to have one or two orders of magnitude more data for training and cross-validating the model.
